## Getting Started

1. Clone the repository
```bash
git clone https://github.com/sai4464/prepnest.git
```

2. Navigate to project directory
```bash
cd prepnest
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

## Deployment
